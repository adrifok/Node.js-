1)Initialize a Node.js project:
npm init => initialize a project of node.js
npm init -y => same accepting all by default
-P --save-prod: package will appear in your dependencies. This is the default.
-D --save-dev: package will appear in your dependencies. 
-O --save-optional: package will appear in your optionalDependencies.
